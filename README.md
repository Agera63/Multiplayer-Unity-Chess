# Chess Game

A chess game for Windows supporting three game modes: play against an AI, play locally with a friend, or challenge someone online!

---

## Requirements

- **Windows** 10 or later
- The game folder must stay **intact** — do not move or delete any files inside it

---

## How to Play

1. Open the game folder and launch **Chess.exe**
2. From the main menu, choose your game mode:

### 🤖 Play vs Bot
- Click **Play vs Bot**
- Choose to play as **White**, **Black**, or **Random**
- The bot will automatically make its moves — just wait for your turn!

### 🌐 Play Online
- **To host:** Click **Play Online** → **Host** — a join code will appear on screen, share it with your opponent
- **To join:** Click **Play Online** → **Join** → enter the code your opponent shared and click **Join**
- Colors are assigned randomly at the start of the game
- ⚠️ Both players need a stable internet connection

---

## Controls

| Action | Input |
|---|---|
| Select a piece | Left click |
| Move a piece | Left click on a highlighted tile |
| Pause / Unpause | Escape |
| Exit to main menu | Pause → Exit |

---

## Pawn Promotion

When one of your pawns reaches the other end of the board, a promotion menu will appear. Select the piece you want to promote to and the game will continue automatically.

---

## Troubleshooting

- **The game won't launch** — Make sure no files have been moved or deleted from the game folder
- **The AI isn't moving** — Make sure the `StreamingAssets` folder is present and intact inside the game folder
- **Online game won't connect** — Check your internet connection and make sure the join code was entered correctly
